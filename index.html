Como integrar com a Lever

1. Acesse as configurações da sua conta Lever

2. Vá para a seção "Integrações"

3. Gere uma API Key com permissões de agendamento

4. Use o webhook URL: https://api.lever.co/v1/opportunities/{id}/addToStage

Convite enviado para o candidato com sucesso!
